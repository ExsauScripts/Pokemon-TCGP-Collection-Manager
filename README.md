# Pokemon TCGP Collection Manager

Web app for managing Pokemon TCG Pocket card collections.
Built with Python (Flask) + SQLite. Developed as a custom client project.

## Features

- Card registry with image, type, abilities, collection and pack
- Real-time quantity updates synced to database (no cache)
- Visual filter — unowned cards shown with gray overlay
- Multi-account support (Main / F2P profiles)
- Card variants for alternate arts (Full Art, etc.)
- Advanced filtering by name, type, rarity, HP, weakness and more
- Full CRUD for cards, packs, collections, types and weaknesses
- Dynamic dropdowns loaded from database

## Tech Stack

- Python / Flask
- SQLite
- HTML / CSS / JavaScript
- Choices.js for multi-select filters
